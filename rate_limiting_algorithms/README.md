# Rate Limiter Project

## Overview

This project implements two rate limiting algorithms:

1. **Token Bucket** - supports bursts of traffic.
2. **Leaky Bucket** - smooths traffic to a fixed rate.

## Files

- `token_bucket.py` - Token Bucket class
- `leaky_bucket.py` - Leaky Bucket class
- `test_rate_limiter.py` - Test script simulating requests

## How to Run

```bash
python test_rate_limiter.py
```
